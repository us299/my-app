About HERITAGE EMPORIUM

Grayscale is a premium quality, free Bootstrap 3 theme created by Start Bootstrap. It can be yours right now, all you need to do is download the template on the preview page. You can use this template for any purpose, personal or commercial.

This striking, black and white theme features stock photographs by Gratisography along with a custom Google Map skin courtesy of Snazzy Maps.

With this template, just the slightest splash of color can make a huge impact on the overall presentation and design.

START USING HERITAGE EMPORIUM!

The great thing about Heritage Emporium is that you can start off right now, and start building your network of story-tellers!

What do I need?

All you need is a server that supports PHP 4.3, MySQL4.1.2 and our package installer. Yes, it is that easy. Heritage Emporium uses wordpress, so the great thing is that you can learn within known territories, win win?

Android?

You heard right! An Android app, so you can add your favorite places from anywhere you are.

But Data connection?

Of course not! Heritage Emporium allows you to store locally and sync with the web interface as and when internet is available. There is nothing coming in the way of your story!

So, what do I do then?

Its simple! Download Heritage Emporium V1.0 that includes a Heritage Emporium (powered by Wordpress) package installer and our .apk android app! Upload the package installer to your workspace, and simply run '/index.php', everything else will be taken care of!
